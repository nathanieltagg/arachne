(In fact, nearly any browser that is not Microsoft Internet Explorer will work.)

Technical details: You're probably using Internet Explorer. Microsoft has been very slow to adopt the HTML5 draft standard, in particular the 'canvas' tag, a requisite feature for Arachne to draw custom diagrams to your screen. Moreover, Microsoft continually and intentionally follows different technical standards for many different browser features. Although many professional sites have programmers and resources to work around the eccentricities, this isn't possible for us.
(Besides, the other browsers are all better.)
