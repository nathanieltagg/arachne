Navigation

Here are some useful keys to navigate Arachne quickly

  • n Next gate
  • p Previous gate
  • + Next slice
  • - Previous slice
  • a All hits in the gate
  • 0 or o Show 'orphans' (all hits that are members of no slice, i.e. slice zero)
  • 1-9 Go to that slice
  • < Previous slice or last slice in previous gate
  • > Next slice or first slice in next gate
